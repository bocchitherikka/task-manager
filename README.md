# Проект: task-manager

## Установка:

```bash
# Клонируем репозиторий
git clone https://github.com/bocchitherikka/task-manager.git

# Пере
cd task-manager/task_manager

docker-compose up -d

docker exec -it task_manager-web python manage.py migrate 
```

## Использование
1. Зарегистрируйтесь и войдите в систему.
2. Создайте новую задачу или присоединитесь к существующей.
3. Управляйте своими задачами и взаимодействуйте с участниками.
